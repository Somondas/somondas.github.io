Virtual desktop



The virtual desktop feature is like from the future. With virtual desktops, you create multiple, separate desktops that each can display different open windows and apps. Multiple desktops are great for keeping unrelated, ongoing projects organized, or for quickly switching desktops before a meeting or a presentation.

To create multiple desktops, click on the "Task View" button on the taskbar,

Click on the "New desktop" button in the top-left corner. Each virtual desktop you create allows you to open different programs. Windows 10 allows you to create an unlimited number of desktops so you can keep track of each one in detail. Each time you create a new desktop, youâ€™ll see a thumbnail of it at the top of your screen in Task View.

Use Start + Ctrl+ D shortcut to create a new virtual desktop without using the Task View.

How to switch between virtual desktops?

To switch between virtual desktops, click on the "Task View" button on the taskbar or press the Start + Tab shortcut.

And select the desktop you want from the top left corner.

You can also use the Start + Ctrl + Left and Start + Ctrl + Right to switch from one desktop to another.
